# 3Sum (Two Pointers)

## 1. Problem Statement
Given an array `nums`, return all unique triplets `[a, b, c]` such that `a + b + c = 0`.

> See full solution in `01-Arrays/13_3sum.md`.
> This file focuses on the Two Pointer pattern specifically.

## 2. Example
```
Input:  nums = [-1, 0, 1, 2, -1, -4]
Output: [[-1,-1,2],[-1,0,1]]
```

## 3. Two Pointer Approach

**Key Idea:**
- Sort the array.
- Fix one element at index `i`, then use two pointers `left` and `right` to find pairs summing to `-nums[i]`.
- This reduces the 3-element search to a 2-element search on a sorted subarray.

```python
def threeSum(nums):
    nums.sort()
    result = []

    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue  # skip duplicates for first element

        left, right = i + 1, len(nums) - 1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total == 0:
                result.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left+1]:
                    left += 1
                while left < right and nums[right] == nums[right-1]:
                    right -= 1
                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1

    return result
```

- **Time Complexity:** O(n²)
- **Space Complexity:** O(1)
